# wx-mall

一个包含后台管理以及微信小程序商城的项目

## 技术点

- java
  - springboot
  - 后台暂定jesite开源框架 （ https://jeesite.gitee.io/ ）
- 数据库
  - mysql
  - redis
  - 搜索引擎es
- 前端
  - vue（react）
